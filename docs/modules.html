Modulok¶

A modulok egy Ilx alkalmazás alap építőelemei. Egy modul egy jól körülhatárolható funkcionalitást valósít meg, ami könnyen beilleszthető az alkalmazásunkba.

Az alábbiakban az Ilx által biztosított modulok részletes leírását találod, illetve a Saját modul fejlesztése oldalon egy saját modul definiálásának a lépéseit mutatjuk be.
